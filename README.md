## Features
- Install essentials via Homebrew
- Leverages the Fish Shell
- Uses Symbolic Links to connect .files

## Mac instructions

1. Install Homebrew: `./brew.sh`
2. Install Fish shell: `./fish.sh`
3. Link Dotfiles: `./link-dotfiles.sh`
